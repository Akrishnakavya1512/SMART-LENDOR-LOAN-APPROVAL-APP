Architecture

Applicant data flows from the Flask form into the preprocessing pipeline, then into the saved classifier. The result returns as an approval or high-risk decision with model confidence.

  1. User layer submits applicant details
  2. Frontend templates collect structured inputs
  3. Flask validates and processes the POST request
  4. ML pipeline transforms features and predicts eligibility
  5. Deployment layer serves browser access through IBM Cloud
